Background

Started in networking before automation was an option—CLI, console cables, and physical installation. Now I build the tools I wish I had back then. Service provider background (Cricket Communications, Charter, TW Telecom) gave me BGP, MPLS L3VPN, and carrier-scale operations. Enterprise roles added SD-WAN, security stack integration, and retail network deployments.

I focus on operational problems: discovery, configuration drift, migration validation, compliance auditing. If it's repetitive and error-prone, I automate it.
