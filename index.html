Secure Cartography & Surveyor Suite

Network Discovery, Mapping, and Management Toolkit

A comprehensive Python-based network discovery, mapping, and management toolkit designed for network engineers and IT administrators. Provides robust features for automated device interrogation, network topology visualization, and configuration management while maintaining strict security standards.

Key Features

  • Automated SSH-based discovery with platform auto-detection for Cisco, Arista, and Aruba devices
  • Interactive Mermaid-based network topology visualization with multiple layout algorithms
  • PyQt6-based modern interface with dark/light mode support
  • Multiple export formats: SVG, GraphML for yEd, Draw.io compatible format
  • Master password-based encryption system with machine-specific keyring integration
  • Device inventory parsing using TextFSM templates with multiprocessing support
  • Configuration backup and management with SQLite database integration
  • Interactive topology enhancement with node editing and icon customization
  • Advanced CLI tool for automation and scripting workflows

SecureCartographyVSC

Network Discovery and Visualization for Visual Studio Code

A comprehensive VS Code extension that brings enterprise-grade network discovery, mapping, and visualization capabilities directly into your development environment. Map your entire network infrastructure with a few clicks using pure JavaScript implementation with no external dependencies.

Key Features

  • Pure JavaScript implementation with no external dependencies or commands required
  • Enterprise-grade network discovery using CDP/LLDP protocols
  • Multi-device support for Cisco, Juniper, and other network vendors
  • Interactive topology visualization with comprehensive graphical interface
  • Multiple export formats: Draw.io, GraphML, JSON, and more
  • Flexible authentication support including SSH keys and enable passwords
  • Recursive discovery with configurable hop count limits
  • TextFSM and regex-based parsing for device outputs
  • Detailed device information collection (interfaces, models, OS versions)
  • Configurable exclusion patterns for precise discovery scope control
