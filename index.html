About

Passionate about network observability, automation, and building tools that make network engineers more effective.

As a Senior Infrastructure Platform Engineer at Kentik, I leverage over three decades of network engineering experience to build innovative solutions in network datacenters and automation. My journey has taken me from traditional network engineering to becoming a multi-language, multi-framework developer creating production-grade network automation tools.

My core philosophy: "Work the problem, build the solution, make it better." I focus on creating practical solutions, tooling and implementations that solve real operational problems, not theoretical dashboards. This approach has led to the creation of comprehensive network automation ecosystems that are actively used by network engineers worldwide.

I'm the creator of the VelociTerm ecosystem - a comprehensive suite of network engineering tools including web-based SSH terminal managers, desktop terminals with real-time monitoring, and network topology discovery platforms. My work has gained industry recognition from other FOSS Companies and is actively shared with the network automation community.
